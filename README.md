# Quiniela del Tri — Sorteo de equipos desde Octavos de Final, Mundial 2026

Misma mecánica que la quiniela de 32, pero para arrancar desde los 16 equipos
que llegan a octavos de final.

## Cómo funciona el formato
- Cada entrada cuesta $100 y te asigna uno de los 16 equipos de octavos al azar.
- Una misma persona puede comprar varias entradas (varios equipos).
- Si tu equipo avanza de ronda (Cuartos, Semifinal, Final/3er lugar), se suma
  otro pago de $100 al bote.
- El bote se reparte 60% / 25% / 15% entre los dueños de: el campeón, el
  subcampeón y el ganador del partido por el 3er lugar.

## Primer uso (importante)
1. Entra a `tusitio.netlify.app/admin.html` con tu clave.
2. Renombra los 16 "Equipo N" con los nombres reales clasificados a octavos.
3. En "Pos. real cuadro" pon la posición de cada equipo en las llaves reales
   (1 al 16) — así el cuadro en `/bracket.html` muestra los cruces correctos,
   sin importar el orden en que los vendas.
4. Comparte `tusitio.netlify.app` para que la gente compre y se sortee su equipo.
5. Conforme avancen las rondas reales, actualiza el estatus de cada equipo y
   guarda — el bote y los premios se recalculan solos.

(Resto de instrucciones de despliegue igual que la versión de 32 equipos — ver
el otro proyecto si ya pasaste por ese proceso.)
